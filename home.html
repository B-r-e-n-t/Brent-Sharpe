What I've Built

Find-A-Dinner

Find-A-Dinner is a web application using the Spoonacular API to search and find dinner ideas and recipes using a keywords search. This project uses HTML, CSS, and JavaScript. It works using Fetch() and JSON to receive the recipes and display the recipess images and links on the UI. Click on the image to view the recipe and begin cooking!

Demo / GitHub

Neighbors Pizza and Beer Garden

Neighbors Pizza and Beer Garden is a website for a fake (for now) pizza restaurant. It is a replicate of McDougals Chicken. This project uses HTML, CSS, and vanilla JavaScript. Some of its features are the Google Maps API, auto slideshow of images using JavaScript, and a collage of imags using CSS Grid.

Demo / GitHub

Pig Game

Pig Game is a desktop and tablet dice game. The game was built using HTML, CSS, and vanilla JavaScript. The methods used in the program is mostly event listeners and if/else statements. Click to roll the dice and get as many points as you can collect before rolling a 1. If you roll a run all of your current score is lost. Try it out!

Demo / GitHub

Sharpe Web Designs

Sharpe Web Designs is a fully responsive website for a fake website development agency. It was built with HTML, CSS, and vanilla JavaScript. Some of its features are Flexbox, CSS Grid, a fade-in scrolling animation, and a functioning contact form. It consists of a home, contact, about, and portfolio page.
